# tunnel

tunnel 是一个反向访问工具，在服务端无法提供固定可访问地址的场景下，可将 tunnel-agent 部署在服务端，
tunnel-agent 会与 tunnel-server 建立一条通讯隧道，并由 tunnel-server 对外提供和原服务一致的服务

## 工作模式

```
|--------------|       |--------------|
|    server    | <-X-- |    client    |
|--------------|       |--------------|
```

```
|--------------|       |---------------|
|    server    |       |               |
|--------------|       |               |
       ^               |               |       |--------------|
       |               | tunnel-server | <---- |    client    |
       |               |               |       |--------------|
|--------------|       |               |
| tunnel-agent | ----> |               |
|--------------|       |---------------|
```
